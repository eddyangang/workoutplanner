## Description

  This web application allows the user to log their workout routine by adding exercises or update their previous exercises, with visualization of their routines through the use of charts and graphs. 

  Technologies Used : 

  * HTML
  * CSS
  * JS
  * MongoDB
  * Mongoose
  * NodeJS
